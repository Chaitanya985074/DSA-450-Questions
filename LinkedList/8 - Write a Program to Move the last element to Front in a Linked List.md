# Write a program to Move the last element to Front in a Linked List.	

Difficulty Level : Easy

Given a linked list, move the last node to the front in a given Singly Linked List.

**Examples :**

```
Input: 1->2->3->4->5
Output: 5->1->2->3->4 

Input: 3->8->1->5->7->12
Output: 12->3->8->1->5->7
```
