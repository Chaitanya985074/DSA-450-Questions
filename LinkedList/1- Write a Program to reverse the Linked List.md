# Write a Program to reverse the Linked List. (Both Iterative and recursive)	

Difficulty Level : Easy

Given a linked list of N nodes. The task is to reverse this list.

**Examples :**

```
Input: 1->2->3->4->5->6
Output: 6->5->4->3->2->1

Input: 2->7->8->9->10
Output: 10->9->8->7->2
```
