# Write a program to cyclically rotate an array by one

Difficulty Level : Basic

Given an array, cyclically rotate the array clockwise by one. 

***Examples:***

```
Input:  arr[] = {1, 2, 3, 4, 5}
Output: arr[] = {5, 1, 2, 3, 4}
```
