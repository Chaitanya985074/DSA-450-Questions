# Program to reverse a string (Iterative and Recursive)

Difficulty Level : Easy

Given a string, write a program to reverse it.

**Examples :**

```
Input  : s = "abc"
Output : s = "cba"

Input  : s = "geeksforgeeks"
Output : s = "skeegrofskeeg"
```
